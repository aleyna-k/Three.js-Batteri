# Three.js-Batteri
🥁 3D Batteri Kit An interactive 3D drum kit built with Three.js and Web Audio API no dependencies, no installs, just open and play.
 

🎸 Full 3D drum set — kick, snare, hi-hat, 2x rack toms, floor tom, ride & crash cymbals
🔊 Synthesized drum sounds via Web Audio API (no audio files needed)
🎹 Keyboard playable — mapped to QWERTY keys
🖱️ Clickable — click any drum in the 3D scene to play it
🌀 Auto-rotate camera with manual drag & scroll zoom
🌊 Reverb toggle — convolution reverb synthesized in-browser
🔴 Record & download — record your performance and save as .webm
📊 VU meter — real-time audio level visualization
